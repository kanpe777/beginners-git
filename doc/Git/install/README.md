## 入っているか確かめる

### Gitが入っている状態
    $ git そのままEnter
    usage: git [--version] [--exec-path[=<path>]] [--html-path] [--man-path] [--info-path]
    ...

入ってた人はback

### Gitが入っていない状態
    $ git そのままEnter
    zsh: command not found: git

### インストール
方法は色々

 * brew install git
 * ports install git
 * epkg install git
 * 本家からダウンロードしてインストール

brew, ports, epkg 知らない人は本家からダウンロードしてインストールしよう。
